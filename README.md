# OSS Audit Platform

An AI-powered Open Source Software Audit Platform.

## Planned Features

- SBOM generation using SCANOSS
- CBOM generation using CryptoFinder
- AI-powered analysis with Claude
- Executive Summary
- Compliance Report
- Markdown report generation

## Workflow

Repository
↓
SCANOSS
↓
CryptoFinder
↓
Claude AI
↓
Audit Report

## CI/CD

The platform is designed to run through GitHub Actions.
